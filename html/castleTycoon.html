  The first game that I created in Roblox was Castle Tycoon. I wanted to learn how to use Roblox Studio to be able to make Tycoons and Obbies (Parkour courses). This game aims to build up the droppers that will make you money and then complete the castle. There are a couple of buttons that build different sections of the castle and some buttons that build the droppers. This is the link to the Roblox Castle Tycoon

This is a picture of the plots in the game. I have it set up for 6 players. The game will assign each plot to a player so that they can only purchase buttons on their plot.

This is a picture of the outside of the castle. The walls are 3 different parts and the towers are 4 different parts. Each increase in price so that the tycoon can be harder and more expensive each time you upgrade.

This is a picture of all the droppers and the buttons that are associated with them. The color of the droppers determines the level. This also changes the price of the drop. The first couple gives 5 cash per drop then the last dropper gives 100 cash each drop. Each button has text above it to tell the player what they are buying.
